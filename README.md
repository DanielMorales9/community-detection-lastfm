# Community Detection
Second project of the master course in [Big Data](http://torlone.dia.uniroma3.it/bigdata/) at Roma Tre University

## Dataset 
- [LastFM](https://grouplens.org/datasets/hetrec-2011/) by GroupLens

## Algorithms
- Cosine similarity
- Clique
- K-plex
- Louvain Method

## Technologies
- Python3
- NetworkX
- Java
- MLlib
- Spark
- Cassandra
- Neo4j 3.0
- Gephi
- Docker
